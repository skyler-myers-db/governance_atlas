# North Star Gap Analysis

Each file in this directory is the page-specific visual contract for a
North Star rebuild. The target mockup must be decomposed here before page
implementation starts.

Checklist statuses:

- `[ ]` open
- `[x]` verified
- `[~]` intentionally degraded or unavailable because live data is missing
- `[>]` explicitly deferred with rationale

Rules:

- Structural mockup regions are not optional.
- Thin live data changes panel content, not page shape.
- Functional smoke does not close visual gaps.
- Completion requires a fresh screenshot and side-by-side review.
